# Bazaar - A New Online Marketplace App

### Author
Mazi Ahmed

### Mission
Bazaar is an online marketplace app where users can buy and sell items locally. The platform facilitates easy transactions by offering delivery services via car shipping. Imagine a fusion of Facebook Marketplace and Uber, creating a seamless experience for users looking to engage within their local market.

### Tech Stack
- Backend: Python, Django
- Frontend: Tailwind CSS
- Project Management: Figma, Trello

### Approach Taken
Similar online marketplace apps that I'm frequently browsing, including Facebook Marketplace and Karrot, all utilize a seamless and easy process which I wanted to emulate. Bazaar was developed with a focus on user experience with a clean and responsive design using Tailwind CSS. Django's framework was utilized for handling database operations, user authentication, and Google Maps API.

### Unsolved Problems

- Improving delivery tracking for users.
